## 📺 GhostyStream

Lightweight streaming integration plugin for all types of Minecraft servers.
Supports **Minecraft 1.21 – 1.21.11** (Paper & Spigot).

---

### ✨ Features

* In-game stream notifications
* Automatic live status messages
* Stream announcement system
* Easy integration with streaming platforms
* Creator Codes for Players
* Supports `&` and HEX color codes
* No database required
* Low performance impact
* Fully customizable language files
* English & German support

---

### 🔄 Dependencies

* EssentialsX -> https://modrinth.com/plugin/essentialsx
* Vault -> https://www.spigotmc.org/resources/vault.34315/

---

### 🧾 Commands

| Command               | Description                        | Permission                  |
| ------------          | -----------------------            | -----------------------     | 
| `/stream`             | Start Stream Announcment in Chat   | ghostystream.stream         |
| `/content`            | Opens a GUI with all Creators      | ghostystream.content        |
| `/creatorcode`        | You can add (Only OP) & redeem Creator Codes | ghostystream.creatorcode    |
| `/ghostystream | /gs` | You can use the reload, addcreator, removecreator command | ghostystream.admin    |
| /                     | You can create creator codes        | ghostystream.creatorcode.create

---

### ⚙️ Installation

1. Download the latest release
2. Place the `.jar` into `/plugins/`
3. Restart your server
4. Configure `config.yml`

---

### 🔄 Compatibility

* Paper
* Spigot
